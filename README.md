## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## 💻 Projeto

O Disk-Pizza é uma aplicação de controle de pedidos, onde é possível cadastrar e excluir pedidos e definir para qual cliente irá a pizza 🍕.
